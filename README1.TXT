# AR Resume Builder & Analyzer

##Description 
A hackathon project that allows users to create and analyze resumes in Augmented Reality (AR). Users can view their resumes in 3D, interact with skills, and get basic suggestions for improvement.

## Features
- Build resumes easily using a web interface
- Visualize resumes in Augmented Reality (AR)
- Automatic skill analysis
- Support for GitHub and LinkedIn links

## Technology Stack
- HTML, CSS, JavaScript
- A-Frame & AR.js for Augmented Reality
- Node.js & Express.js (optional backend)
